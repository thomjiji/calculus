### Pythagorean trigonometric identity

$$\sin ^{2} \theta+\cos ^{2} \theta=1$$

### Sum and difference identities for sine and cosine

| Identities                                        |
| ------------------------------------------------- |
| $\sin(a + b)$ = $\sin(a)\cos(b) + \cos(a)\sin(b)$ |
| $\sin(a - b)$ = $\sin(a)\cos(b) - \cos(a)\sin(b)$ |
| $\cos(a + b)$ = $\cos(a)\cos(b) - \sin(a)\sin(b)$ |
| $\cos(a - b)$ = $\cos(a)\cos(b) + \sin(a)\sin(b)$ |
